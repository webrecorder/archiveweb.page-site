Archivador web interactivo basado en navegador, un proyecto de webrecorder.

ArchiveWeb.page es la nueva herramienta de Webrecorder para transformar tu navegador en un sistema interactivo de archivo web con todas las funciones.

ArchviveWeb.page esta disponible como una extensión para cualquier navegador con base Chrome o Chromium. (Una versión en aplicación esta en proceso de desarrollo.)

Para crear archivos web, la extensión (o aplicación) será necesaria. Una vez creados, los archivos pueden ser consultados en cualquier navegador moderno usando ReplayWeb.page -- ¡sin extensión requerida!
